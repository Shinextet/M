Backend Integration

This prototype uses browser localStorage for persistence. To scale to a full backend, connect to the Flask/Django API structure below:

# Flask Backend Structure
# ----------------------------
# app.py         ← Flask app entry
# models.py      ← SQLAlchemy models
#   - User(id, name, email)
#   - Goal(id, user_id, name, target, current)
#   - Transaction(id, user_id, type, amount, note, date)
# routes/
#   - auth.py    ← /api/auth/*
#   - goals.py   ← /api/goals/*
#   - txns.py    ← /api/transactions/*
# config.py      ← DB URI, secret keys
#
# Django alternative:
#   savings/models.py  ← Goal, Transaction
#   savings/views.py   ← DRF ViewSets
#   savings/urls.py    ← Router endpoints
